# Histopathology Molecular Subtype Classification

This project focuses on the prediction of breast cancer molecular subtypes from histopathology whole-slide image (WSI) patches using deep learning techniques. Whole-slide histopathology images are extremely large and heterogeneous, and they often contain staining variability, acquisition artifacts, and noisy backgrounds. These characteristics make direct slide-level training with convolutional neural networks difficult. To address this challenge, this project implements a patch-based learning pipeline that extracts informative regions from the tissue and performs supervised multi-class classification on smaller image patches.

The classification task involves predicting one of four molecular subtypes: Luminal A, Luminal B, HER2-positive, or Triple Negative. Because the evaluation metric is macro F1-score, the models are designed to balance precision and recall rather than focusing only on overall accuracy. The pipeline also considers the presence of noisy or low-quality images and aims to improve model robustness under these conditions.

A preprocessing pipeline was designed to improve input quality and reduce artifacts before training. The pipeline includes artifact detection using HSV color analysis to identify abnormal color patterns such as yellow, green, or cyan artifacts that frequently appear in histopathology images. Images with excessive artifacts are removed after manual inspection. Additionally, strong color artifacts are suppressed by detecting bright or green-stained pixels and replacing them with a neutral gray background with small noise to avoid introducing misleading patterns. The provided binary masks are also used to focus the model on tissue regions of interest. Pixels outside the region of interest are replaced with a gray background rather than black to better resemble the natural slide background and reduce spurious correlations.

To further improve learning from relevant tissue structures, ROI-guided tiling is applied. Candidate tiles are generated around the tissue region using bounding boxes and grid sampling, and a greedy strategy is used to select tiles with high tissue coverage. Tile quality filtering is then applied to discard tiles containing little tissue or low visual information based on variance checks. This process ensures that the training data primarily contains informative tissue patterns.

Two main deep learning architectures are evaluated in this project. The first model is a patch-based classifier built on pretrained ResNet backbones, including ResNet50 and ResNet101. These networks use global average pooling followed by a fully connected classification head to predict the four molecular subtypes. The second architecture is a Dual-Path EfficientNet-B2 model that incorporates both RGB image information and the binary mask. In this model, the RGB branch uses a pretrained EfficientNet-B2 encoder to extract visual texture features, while a lightweight convolutional neural network processes the binary mask to capture spatial information about tissue regions. The features from both branches are fused in latent space and passed through a multilayer perceptron classifier to produce the final predictions.

During training, the models use a multi-class cross-entropy loss function with class weighting to mitigate class imbalance. Optimization is performed using AdamW or Ranger optimizers, and lightweight spatial data augmentation such as random flips and small affine transformations is applied to improve generalization. The input size for all models is 224×224 pixels. Training is performed at the tile level, while evaluation is conducted at the image level by averaging the predicted probabilities from all tiles associated with a single image.

Experimental results show that the patch-based pipeline significantly improves performance compared to non-patching baselines. While ResNet101 achieves the highest validation F1-score, the Dual-Path EfficientNet-B2 model achieves the best test F1-score, indicating better robustness to noisy and heterogeneous data. The results demonstrate that combining RGB visual features with ROI-based spatial information can improve generalization in histopathology image classification tasks.

This repository contains the preprocessing pipeline, model implementations, training scripts, and the project report. The work was completed by the Marcus.ai team as part of the AN2DL challenge and aims to explore robust deep learning methods for histopathology image analysis.
